Twitter / X

Step 1: Create a Developer Account

  1. Go to https://developer.twitter.com/en/portal/dashboard
  2. Sign in with your Twitter account
  3. Apply for a developer account if you don't have one
  4. Choose "Free" tier (allows 1,500 tweets/month)

Step 2: Create a Project & App

  1. Click "Create Project" in the dashboard
  2. Name your project (e.g., "StewardGrowth Marketing")
  3. Select use case: "Making a bot" or "Building tools for Twitter users"
  4. Create an app within the project

Step 3: Configure OAuth 2.0

  1. Go to your App Settings > "User authentication settings"
  2. Click "Set up"
  3. Enable OAuth 2.0
  4. Type of App: "Web App, Automated App or Bot"
Callback URL:
https://stewardgrowth.vercel.app/api/oauth/twitter/callback
Required Scopes: tweet.read, tweet.write, users.read, offline.access

Step 4: Get Your Credentials

  1. Go to "Keys and tokens" tab
  2. Copy your Client ID
  3. Generate and copy your Client Secret
TWITTER_CLIENT_ID=your_client_id_here
TWITTER_CLIENT_SECRET=your_client_secret_here

Meta (Facebook, Instagram, Threads)

One App for Three Platforms: A single Meta app handles Facebook, Instagram, and Threads.

Step 1: Create a Meta Developer Account

  1. Go to https://developers.facebook.com/
  2. Click "Get Started" or "My Apps"
  3. Complete developer registration if needed

Step 2: Create an App

  1. Click "Create App"
  2. Select "Business" as the app type
  3. Name your app (e.g., "StewardGrowth")
  4. Add a contact email

Step 3: Add Products

  1. In App Dashboard, go to "Add Products"
  2. Add "Facebook Login" - click "Set Up"
  3. Add "Instagram Graph API" - click "Set Up"
  4. Add "Threads API" (if available) - click "Set Up"

Step 4: Configure Facebook Login

  1. Go to Facebook Login > Settings
  2. Enable "Client OAuth Login"
  3. Enable "Web OAuth Login"
Valid OAuth Redirect URIs:
https://stewardgrowth.vercel.app/api/oauth/facebook/callback
https://stewardgrowth.vercel.app/api/oauth/instagram/callback
https://stewardgrowth.vercel.app/api/oauth/threads/callback
Required Permissions:
Facebook: pages_manage_posts, pages_read_engagement, pages_show_list
Instagram: instagram_basic, instagram_content_publish, instagram_manage_insights
Threads: threads_basic, threads_content_publish

Step 5: Get Your Credentials

  1. Go to Settings > Basic
  2. Copy the App ID
  3. Click "Show" next to App Secret and copy it
META_APP_ID=your_app_id_here
META_APP_SECRET=your_app_secret_here
Important Requirements:
  • Facebook posting requires a Facebook Page (not personal profile)
  • Instagram requires a Business or Creator account
  • Instagram account must be linked to a Facebook Page
  • App must be in "Live" mode for production use
T

TikTok

Video Only: TikTok only supports video uploads. Text-only posts are not possible.

Step 1: Create a TikTok Developer Account

  1. Go to https://developers.tiktok.com/
  2. Click "Manage apps" or sign up
  3. Complete the developer registration

Step 2: Create an App

  1. Click "Create app"
  2. Choose "Content Posting API" as the product
  3. Fill in app details
  4. Submit for review

Step 3: Configure OAuth

  1. Go to your app's configuration
  2. Add the redirect URI
Redirect URI:
https://stewardgrowth.vercel.app/api/oauth/tiktok/callback
Required Scopes: user.info.basic, video.publish

Step 4: Get Your Credentials

  1. Go to your app's "Manage" page
  2. Copy the Client Key
  3. Copy the Client Secret
TIKTOK_CLIENT_KEY=your_client_key_here
TIKTOK_CLIENT_SECRET=your_client_secret_here
Approval Process: TikTok requires app review before going live. This can take 1-2 weeks.
Y

YouTube

Video Only: YouTube API is for video uploads. Community posts require different permissions.

Step 1: Create a Google Cloud Project

  1. Go to https://console.cloud.google.com/
  2. Click "Select a project" > "New Project"
  3. Name your project (e.g., "StewardGrowth")
  4. Click "Create"

Step 2: Enable YouTube Data API

  1. In the Google Cloud Console, go to "APIs & Services" > "Library"
  2. Search for "YouTube Data API v3"
  3. Click on it and click "Enable"

Step 3: Create OAuth Credentials

  1. Go to "APIs & Services" > "Credentials"
  2. Click "Create Credentials" > "OAuth client ID"
  3. Configure consent screen first if prompted:
    • User Type: External
    • Add app name, user support email, developer email
    • Add scopes: youtube.upload, youtube.readonly
  4. Application type: "Web application"
  5. Name: "StewardGrowth"
Authorized redirect URI:
https://stewardgrowth.vercel.app/api/oauth/youtube/callback
Required Scopes:
https://www.googleapis.com/auth/youtube.upload
https://www.googleapis.com/auth/youtube.readonly

Step 4: Get Your Credentials

  1. After creating, you'll see your Client ID
  2. Click "Download JSON" or copy the Client Secret
GOOGLE_CLIENT_ID=your_client_id.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=your_client_secret_here
Publishing Status: While in "Testing" mode, only test users you add can connect. Submit for verification to allow all users.
P

Pinterest

Step 1: Create a Pinterest Developer Account

  1. Go to https://developers.pinterest.com/
  2. Sign in with your Pinterest account
  3. Click "My apps"

Step 2: Create an App

  1. Click "Create app"
  2. Fill in app name and description
  3. Select your use case

Step 3: Configure OAuth

  1. Go to your app settings
  2. Add the redirect URI
Redirect URI:
https://stewardgrowth.vercel.app/api/oauth/pinterest/callback
Required Scopes: pins:read, pins:write, boards:read, boards:write, user_accounts:read

Step 4: Get Your Credentials

  1. In app settings, find your App ID
  2. Generate and copy your App Secret
PINTEREST_APP_ID=your_app_id_here
PINTEREST_APP_SECRET=your_app_secret_here
V

Adding Variables to Vercel

Step 1: Access Vercel Dashboard

  1. Go to https://vercel.com/dashboard
  2. Select the "stewardgrowth" project

Step 2: Add Environment Variables

  1. Go to "Settings" > "Environment Variables"
  2. Add each variable one at a time
  3. Select environments: Production, Preview, Development

All Required Variables

# Twitter/X
TWITTER_CLIENT_ID=
TWITTER_CLIENT_SECRET=

# Meta (Facebook, Instagram, Threads)
META_APP_ID=
META_APP_SECRET=

# TikTok
TIKTOK_CLIENT_KEY=
TIKTOK_CLIENT_SECRET=

# YouTube (Google)
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

# Pinterest
PINTEREST_APP_ID=
PINTEREST_APP_SECRET=

Step 3: Redeploy

  1. After adding variables, go to "Deployments"
  2. Click the three dots on the latest deployment
  3. Select "Redeploy"
Testing: After setup, go to /ads in StewardGrowth and try connecting each platform.
